01 Cloud Security

Designing a Lightweight Misconfiguration Monitoring Approach for Secure Cloud Environments

A low-overhead approach for continuously checking cloud resources — IAM policies, storage buckets, network rules — against secure baselines, surfacing risky misconfigurations before they become breaches, without the cost of heavyweight scanning tools.

Highlights
  • Continuous baseline checks against security best practices (CIS-style benchmarks)
  • Lightweight agent with minimal compute and cost footprint
  • Prioritised, actionable findings with clear remediation guidance
Python Cloud APIs Policy-as-Code CIS Benchmarks

02 Host Security

Design and Implementation of a File Integrity Monitoring (FIM) System with Tamper Alerts

A file integrity monitoring system that fingerprints critical files using cryptographic hashes, detects unauthorised additions, edits or deletions in real time, and raises immediate tamper alerts so defenders can respond before damage spreads.

Highlights
  • Cryptographic hash baselining (SHA-256) for trusted file state
  • Real-time change detection with full audit logging
  • Instant tamper alerts via email / messaging on suspicious changes
Python SHA-256 Real-time Monitoring Alerting

03 Machine Learning

Development of a Network Intrusion Detection Model Using Machine Learning

A machine-learning model that classifies network traffic as benign or malicious by learning from labelled flow data. It combines feature engineering with supervised classifiers to detect intrusions and attack patterns with high accuracy and low false positives.

Highlights
  • Trained on standard intrusion datasets (e.g. NSL-KDD / CICIDS)
  • Feature engineering and model tuning for accuracy vs. false alarms
  • Evaluated with precision, recall, F1 and confusion-matrix analysis
Python scikit-learn Random Forest / XGBoost Pandas
